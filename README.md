# Bunker Game Frontend

A React + TypeScript + Vite frontend for the Bunker party/survival game, utilizing Socket.io for real-time multiplayer gameplay.

## Tech Stack
- **React** (v18)
- **TypeScript**
- **Vite**
- **TailwindCSS**
- **Socket.io Client**
- **Lucide Icons**

## Getting Started

### Installation
```bash
npm install
```

### Run in Development Mode
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
